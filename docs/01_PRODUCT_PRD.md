# AI Canvas v1.0 产品需求文档

## 1. 产品定位

AI Canvas 是一个本地优先的无限画布 AI 工作流应用。

产品参考：

- Figma
- FigJam
- ComfyUI
- Lovart
- Linear
- Raycast

核心形态：

```text
无限画布 + 节点系统 + AI 工作流
```

第一版目标不是生成图片，而是先完成：

```text
框架 + 美工 + 基础交互 + 节点系统
```

## 2. 第一版范围

第一版只做本地可运行 Web 原型：

- Next.js 本地运行
- 高保真 UI 框架
- 无限画布
- 节点组件
- 连线系统
- 网格系统
- 左侧工具栏
- 顶部工具栏
- 右下角缩放/网格控制
- 设置页空壳
- API 配置预留

## 3. 第一版不要做

不要开发：

- 登录
- 云端保存
- 导出
- 分享
- 支付
- 团队协作
- 真实 AI 生成
- 复杂项目管理

## 4. 后期功能方向

后期再加入：

- AI 图片分析
- Prompt 优化
- 多图生成
- 图片编辑
- 视频生成
- 项目保存
- 模板库
- API 用量统计
- 桌面端 Tauri 打包
