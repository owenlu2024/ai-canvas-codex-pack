# AI Canvas 交互规范 v1.0

## 1. 画布交互

必须支持：

```text
鼠标中键按住拖动画布
鼠标中键按住时，光标显示为手掌抓取状态
松开鼠标中键后，光标恢复为普通点选状态
普通滚轮上下滚动画布
Shift + 滚轮横向移动
Command + 滚轮缩放
Ctrl + 滚轮缩放
触控板双指移动
触控板双指缩放
```

缩放范围：

```text
10% - 400%
```

## 2. 网格交互

只有一个网格开关。

```text
网格开 = 显示点阵网格 + 节点拖动吸附
网格关 = 隐藏点阵网格 + 节点自由拖动
```

网格必须无限延伸。

## 3. 节点选择

```text
点击节点 = 单选
Shift + 点击节点 = 增加或取消选择
Esc = 取消所有选择
空白处按住左键拖拽 = 框选多个节点
空白处双击左键 = 在双击位置打开添加节点菜单
```

光标规则：

```text
鼠标在节点外 = 点选光标
鼠标在节点上 = 手指点击光标
拖动节点 / 已框选节点 = 手掌抓取光标
鼠标在文本输入区域 = 文本输入光标
```

选中节点可以：

```text
一起拖动
一起删除
一起群组
```

## 4. 图层规则

必须实现 zIndex 管理。

核心原则：

```text
谁最后被点击，谁就在最上层。
```

具体要求：

```text
点击节点时，节点自动置顶
拖动节点时，节点自动置顶
选中节点不能被未选中节点遮挡
多选节点拖动时，整组选中节点整体置顶
群组后，群组作为整体图层管理
每个节点保存 zIndex
每次点击或拖动节点：globalZIndex += 1; node.zIndex = globalZIndex
渲染节点时按 zIndex 从小到大排序
```

## 5. 节点删除

删除方式：

```text
选中节点后点击左侧垃圾桶
按 Delete / Backspace
```

删除节点时，必须同时删除与该节点相关的所有连接线。

## 5.1 撤销 / 重做

顶部 Undo / Redo 必须有真实作用。

```text
记录最近 10 步画布内容操作。
支持点击顶部 Undo / Redo。
支持 Cmd/Ctrl + Z 撤销。
支持 Cmd/Ctrl + Shift + Z 重做。
记录范围：添加节点、删除节点/连接线、连接、拖动节点、群组、取消群组、清空节点内容、文本编辑开始前状态。
不记录范围：缩放、平移、网格开关、选中态变化。
```

## 5.2 工作区保存 / 恢复

必须支持：

```text
前端刷新后恢复上次工作区
dev server 重启后恢复上次工作区
启动端口变化后恢复上次工作区
顶部项目名下拉菜单提供“保存”
```

保存内容：

```text
项目名
节点
连接线
节点内容
节点位置
节点层级
画布视口和缩放
网格状态
```

本机工作区文件格式：

```text
.aicanvas
```

## 6. 群组 / 取消群组

左侧工具栏提供：

```text
Group
Ungroup
```

选中多个节点后点击 Group，创建视觉群组。

取消群组后节点恢复独立。

## 7. 添加节点

点击左侧添加节点按钮，不要立即创建节点。

应弹出添加节点面板：

```text
搜索节点
Image
Prompt
Image Chat
Multi Generate
```

选择节点后，在画布中心或当前鼠标附近创建节点。

后期支持：

```text
双击画布空白处打开添加节点面板
从连接点拖到空白处，弹出可连接节点列表
```

## 7.1 Generate Image 节点

Generate Image 是 AI 成图节点。

端口：

```text
左侧绿色 = 图片输入
左侧黄色 = 文本输入
右侧绿色 = 图片输出
左侧绿色 / 黄色输入端口必须一直显示，不能因为未连接而隐藏。
```

模型：

```text
只显示 3 个成图模型
gpt-image-2
gemini-3.1-flash-image-preview
gemini-3-pro-image-preview
```

参数面板：

```text
切换模型后，读取对应模型规格并显示不同参数。
AR 默认 Auto，比例选项使用英文：
Auto / 1:1 Square / 2:3 Portrait / 3:2 Landscape / 3:4 Portrait / 4:3 Landscape / 4:5 Portrait / 5:4 Landscape / 9:16 Phone Portrait / 16:9 Widescreen / 21:9 Ultrawide / 4:1 Superwide / 1:4 Supertall / 8:1 Extreme Wide / 1:8 Extreme Tall
GPT Res 支持：1K / 2K / 4K
Gemini Res 支持：512 / 1K / 2K / 4K
gpt-image-2：AR / Res / Quality / Image Count
gemini-3.1-flash-image-preview：AR / Res / Image Count
gemini-3-pro-image-preview：AR / Res / Image Count
Thinking Mode、Google Search、Image Search 只有在异步图片任务文档明确给出对应 input 字段后才展示。
```

运行：

```text
点击 Run 后执行图片生成。
AI 返回图片前，Generate Image 节点边框保持蓝紫色灯带逆时针循环移动；真实 AI 接入后，无论等待多久都保持循环，直到接口返回或失败。
AI 返回图片并创建输出 Image 节点后，Generate Image 节点边框恢复正常。
支持文生图：只连接文本输入。
支持图成图：只连接单个图片输入。
支持多图参考：连接多个图片输入。
支持图文生图：同时连接图片输入和文本输入。
Image Count 决定生成结果张数，生成成功后在节点右侧自动创建对应数量的 Image 节点。
同一个 Generate Image 再次 Run 时，不能自动删除、覆盖或替换之前生成的输出 Image 节点。
再次 Run 的新输出必须追加到已有输出旁边，保持间距，不能互相重叠。
已生成的输出 Image 节点只能由用户手动删除。
每个结果 Image 节点都从 Generate Image 的右侧绿色输出点连接到 Image 节点左侧绿色输入点。
```

## 7.2 Image 节点编号与导入

Image 节点必须显示统一编号。

```text
编号范围：001 - 100。
Image 图框数量上限：100 个。
编号位置：Image 节点顶部标题栏居中。
编号颜色：Selected Purple Blue #6C63FF。
手动添加的 Image 节点和 AI 生成的 Image 节点使用同一套编号。
删除 Image 节点后，已有节点编号不变化。
新增 Image 节点时，自动使用 001-100 中当前缺失的最小编号。
当前已达到 100 个 Image 图框时，不再创建新的 Image 图框。
编号用于多图 AI 工作流中清晰标识每张图片。
```

图片拖拽导入：

```text
拖拽图片到已有空白 Image 节点内时，图片加载到该 Image 节点，不新建节点。
拖拽图片到画布空白处时，自动新建 Image 节点。
```

图片粘贴导入：

```text
复制图片后，如果当前单选了一个空白 Image 节点，粘贴时图片加载到该 Image 节点。
复制图片后，如果没有单选空白 Image 节点，粘贴时自动新建 Image 节点。
```

## 8. 连接交互

节点前后都有圆点。
连接必须从圆点拖出。

端口方向：

```text
右侧圆点 = 输出端，只能从这里拖出连接。
左侧圆点 = 输入端，只接收对应颜色的连接。
普通未连接节点的左侧输入圆点默认隐藏。
当有连接进入该节点左侧输入端时，左侧输入圆点显示。
断开该输入连接后，左侧输入圆点再次隐藏。
Generate Image 节点的左侧绿色 / 黄色输入圆点必须一直显示。
```

颜色规则：

```text
绿色 = Image
黄色 = Text
```

连接规则：

```text
绿色只能连接绿色
黄色只能连接黄色
不同颜色禁止连接
```

拖线时：

```text
同色点高亮
不同色点显示错误状态
松手后不创建连接线
```

连接线必须从圆点中心连接到圆点中心。

## 9. 连接线删除

点击连接线后：

```text
连接线高亮
连接线中间显示红色减号按钮
```

点击红色减号：

```text
立即删除连接线
```

也支持：

```text
选中连接线后按 Delete / Backspace 删除
```

## 10. 图片拖入和粘贴

支持拖图片文件到画布空白处：

```text
自动创建 Image 节点
图片自动放入节点
```

支持复制图片后粘贴：

```text
Command + V / Ctrl + V
自动创建 Image 节点
```

支持格式：

```text
PNG
JPG
JPEG
WEBP
```

多张图片拖入：

```text
自动创建多个 Image 节点
按网格排列
```

Image 节点图片框：

```text
鼠标移入图片框 = 手指点按光标
Image 节点右上工具区显示下载图标和清空图标。
有图片时点击下载图标 = 将当前图框内图片保存到本地下载目录。
空图片框下载图标置灰不可用。
空图片框双击 = 打开本地图片选择器
已有图片时双击图片框 = 在画布右侧独立区域显示完整图片预览
图片预览区滚轮 = 以光标为中心，在 1x - 4x 之间细步进丝滑放大 / 缩小预览图，不影响画布缩放
图片预览区中键拖动 = 平移预览图，不影响画布平移
单击画布 / 节点 / 预览区域 = 关闭图片预览
选择图片后 = 图片写入当前 Image 节点，并记录一步撤销历史
图片显示 = 必须完整显示，横图和竖图都使用 contain，不裁切
```

## 11. 不显示常驻提示

画布上不要显示：

```text
拖图片到画布空白处
复制粘贴图片
按 Shift 多选
按 Esc 取消
```

界面必须干净，像专业工具。
