# AI Canvas 第一版验收清单

## UI

- [ ] 整体风格接近 references 中高保真参考图
- [ ] 不是后台系统风格
- [ ] 顶部栏极简
- [ ] 左侧工具栏正确
- [ ] 右下控制栏正确
- [ ] 节点圆角、阴影、字体接近规范
- [ ] 画布没有常驻新手提示文字

## 画布

- [ ] 无限画布可拖动
- [ ] 中键拖动画布可用
- [ ] Cmd/Ctrl + 滚轮缩放可用
- [ ] 缩放范围 10%-400%
- [ ] 网格无限延伸
- [ ] 网格开关同时控制吸附
- [ ] 网格永远在最底层

## 节点

- [ ] 可以添加 Image 节点
- [ ] 可以添加 Prompt 节点
- [ ] 可以添加 Image Chat 节点
- [ ] 可以添加 Multi Generate 节点
- [ ] 节点可拖动
- [ ] 点击节点置顶
- [ ] 选中节点不会被遮挡
- [ ] Delete 可删除节点
- [ ] 删除节点同时删除相关连线

## 选择

- [ ] 点击节点单选
- [ ] Shift + 点击多选/取消选择
- [ ] Esc 取消选择
- [ ] 框选可用
- [ ] 多选节点可一起拖动

## 连接

- [ ] 绿色点只能连绿色点
- [ ] 黄色点只能连黄色点
- [ ] 不同色禁止连接
- [ ] 连接线从圆点中心到圆点中心
- [ ] 点击连接线显示红色减号
- [ ] 点击红色减号可删除连接线

## 图片导入

- [ ] 拖图片到画布自动创建 Image 节点
- [ ] 粘贴图片自动创建 Image 节点
- [ ] 多张图片拖入自动创建多个 Image 节点

## 暂不做

- [ ] 不做登录
- [ ] 不做保存
- [ ] 不做分享
- [ ] 不接真实 AI
